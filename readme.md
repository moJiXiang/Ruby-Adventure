游戏策划：

场景内有树，有箱子，Player可以通过砍树收集木材，打开箱子收集齿轮子弹

场景内有两种机器人，一种是被邪恶轻度污染的，能够被一颗齿轮子弹修复，一种是被邪恶重度污染的机器人，需要三颗齿轮子弹才能修复。

修复完的机器人需要被荷花净化，才能被委派任务，比如砍树收集木材。

收集的木材能够制作栅栏作为防御。

两种被污染的机器人会从三个方向不断的产生并破坏房子（房子有血条）

Player需要做的是，收集木材制作栅栏防御，收集宝箱（做好的栅栏或者齿轮子弹），制作木桥收集荷花净化（切换子弹）被污染的机器人。

进入房子（进入另外一个场景）找到青蛙大师发动终极技能，清除所有被污染的机器人。也可以找青蛙大师用材料置换物品。
